◆ setParams()

InfluxDB2\Model\Query::setParams ( $params)

Sets params

Parameters
map[string,object]|null$params Key-value pairs passed as parameters during query execution. To use parameters in your query, pass a query with params references (in dot notation)–for example: json query: "from(bucket: params.mybucket) |> range(start: params.rangeStart) |> limit(n:1)" and pass params with the key-value pairs–for example: json params: { "mybucket": "environment", "rangeStart": "-30d" } During query execution, InfluxDB passes params to your script and substitutes the values. #### Limitations - If you use params, you can't use extern.
